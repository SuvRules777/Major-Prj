Methodology: This system uses a YOLOv8n object detection model trained on 1,445 annotated fish images (80/10/10 train-val-test split). Fish length is estimated from bounding box pixel dimensions scaled via the px/cm parameter. Biomass is computed using the standard allometric equation W = a × Lb where a = 0.01, b = 3.0. Final test set performance: Precision 93.8%, Recall 94.4%, mAP@50 94.9%.
